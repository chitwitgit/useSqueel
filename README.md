# useSqueel Monorepo

This is a monorepo housing the promotional page for the `useSqueel` package and the package itself.

## Package Structure

- **Promotional Page**: This React + Vite application showcases the `useSqueel` package
- **Package**: The actual `useSqueel` package is located in the [`packages/useSqueel`](packages/useSqueel) directory

For detailed information about the `useSqueel` package, please refer to its [README](packages/useSqueel/README.md).
